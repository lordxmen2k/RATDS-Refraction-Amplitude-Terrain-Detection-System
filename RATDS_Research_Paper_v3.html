Refraction Amplitude Terrain Detection System (RATDS)

A Physics-Driven Fractal Framework for Material Recognition from RGB Images

Author: Gerald Enrique Nelson Mc Kenzie
DOI: 10.5281/zenodo.20057189
ORCID: 0009-0008-6017-4922
Date: 05-04-2026
Abstract. We present the Refraction Amplitude Terrain Detection System (RATDS), a material recognition framework that reframes image patches as terrain amplitude signals and applies seismic refraction physics combined with fractal dimension theory to identify material composition from standard RGB images. The system extracts a 14-dimensional feature vector from each image patch through five independent fractal estimators — Variance Fractal Dimension (VFD), the Mandelbrot-Richardson divider method, box-counting, correlation dimension, and 2D radial spectral β estimation — applied to illumination-normalized RGB channels. A novel set of spectral signature features, including spectral flatness and per-channel β ratios, enables discrimination between materials with similar roughness but different wavelength-dependent reflectance properties. Material profiles are stored as external JSON files and can be empirically calibrated from photographs of known materials without any code modification. The system operates with zero labeled training data in rule-based mode and achieves correct top-1 classification of raw silver and gold ore photographs using spectral flatness as the primary discriminator (σβ ≈ 0.01 for silver vs. σβ ≈ 0.14 for gold). All computations run on CPU at approximately 13 ms per 128×128 patch.

1. Introduction

Material recognition from photographs is a problem that resists simple solutions. The same material — metal, glass, cotton — can look entirely different under different lighting, at different scales, from different viewpoints, or when formed into different objects. Conversely, different materials can share color, brightness, or gross shape while differing only in fine surface texture. This fundamental ambiguity has driven the field toward increasingly data-hungry approaches: modern convolutional neural networks trained on millions of labeled pixels routinely exceed 80% accuracy on standard benchmarks, approaching human-level performance.

Yet this accuracy comes with costs that are prohibitive in many real-world settings. Large labeled datasets are expensive to construct. Trained models are opaque — they produce classifications without any physical explanation of why a given material was identified. Adding a new material class requires retraining. And the learned representations have no grounding in the physical properties of surfaces.

RATDS takes a different starting point. Rather than learning what materials look like, it starts with what materials are physically: surfaces characterized by roughness, self-similarity, and spectral reflectance properties that determine how they scatter and reflect light. These properties map directly to fractal mathematics. A smooth polished metal surface has a high Hurst exponent (H → 1), steep spectral amplitude decay (β > 3), and low fractal dimension (D < 1.3). A cotton or cardboard surface has a low Hurst exponent (H → 0), flat spectrum (β < 2), and high fractal dimension (D> 1.7).

Key Insight: Spectral Signatures

Different metals have characteristic wavelength-dependent reflectance: silver is nearly spectrally neutral, gold strongly absorbs blue light, and copper moderately absorbs blue while enhancing red. These differences appear as measurable variation in the per-channel spectral exponents βR, βG, βB.

2. Mathematical Framework

2.1 2D Radial Power Spectrum

For a p×p normalized patch, we compute the 2D discrete Fourier transform with Hann windowing:

W(x, y) = A_norm(x, y) · w_H(x) · w_H(y)
F(u, v) = Σ_x Σ_y W(x, y) · exp(−2πi(ux + vy)/p)
P(u, v) = |F(u, v)|²

The radial average bins power by frequency magnitude r = √(u² + v²). The spectral exponent β₂d is estimated by log-log linear regression:

log P_radial(r) = C − β₂d · log r
β̂₂d = −slope

For isotropic fBm surfaces, β₂d = 2H + 2 = 6 − 2D.

2.2 Spectral Signature Features

Feature Definition Physical Meaning
σ_β std(β_R, β_G, β_B) Spectral flatness — low for neutral metals, high for blue-absorbing metals
ρ_B β_B / mean(β) Blue-channel ratio — suppressed for gold, equal for silver
β_spread max(β) − min(β) Total spectral variation across channels

2.3 Illumination Normalization

Real photographs contain low-frequency shading gradients. We remove these via high-pass filtering:

A_low(x, y) = G_σ * A_L(x, y)          # σ = 0.25 · patch_size
A_norm(x, y) = A_L(x, y) − A_low(x, y)

3. Material Classification

3.1 JSON Profile System

Material profiles are stored as external JSON files in a profiles/ directory. Each profile contains feature ranges (5th–95th percentile), calibration metadata, synthetic parameters, and display properties. Profiles are auto-discovered at runtime. Adding a new material requires only creating a new JSON file — no code modification.

3.2 Background Removal

For product photography with uniform backgrounds, the system automatically detects and masks near-uniform regions before patch extraction. Only foreground patches are analyzed, preventing background pixels from corrupting the median aggregation.

4. Experiments

6. Discussion

6.1 Why Spectral Signatures Work

Silver and gold ores have nearly identical roughness, fractal dimension, and contrast. They differ primarily in their electronic structure: gold has a filled d-band that creates strong interband absorption in the blue region of the visible spectrum (λ < 500 nm), while silver's d-band is deeper in energy, resulting in nearly flat reflectance across the visible range. These physical differences manifest as measurable differences in the per-channel power spectra of reflected light.

6.2 Background Removal Impact

Product photography with uniform backgrounds presents a unique challenge: the background is often 80–95% of the image area. Without background removal, median aggregation is dominated by uniform white patches, producing β ≈ 2.0 (flat spectrum) and near-zero contrast — features characteristic of paper or cardboard, not metal. The auto-masking system detects uniform regions via local variance thresholding and analyzes only foreground patches, restoring correct material detection.

6.3 Limitations

  • Profile calibration dependency: Estimated profiles may have ranges that are too wide or too narrow. Calibrated profiles are marked in JSON metadata.
  • Mid-range material confusion: Plastic, leather, rubber, and wood have overlapping feature ranges.
  • Illumination color casts: Strong color casts can affect per-channel β estimates.
  • fBm model assumption: Surfaces with periodic structure or sharp edges deviate from fractional Brownian motion.

7. Conclusion

RATDS presents a physically grounded alternative to black-box deep learning for material recognition from RGB images. By treating image patches as seismic amplitude signals and extracting five independent fractal dimension estimates from their spectral and geometric properties, the system achieves material classification without learned representations. The novel spectral signature features — spectral flatness and per-channel β ratios — enable discrimination between materials with similar roughness but different wavelength-dependent reflectance, such as silver and gold ores.

The external JSON profile system makes the system extensible: adding a new material requires only running the calibration tool on photographs, with no code modification. The system runs at approximately 13 ms per 128×128 patch on a single CPU core, requires no GPU, and provides fully interpretable classifications grounded in physical surface properties.
